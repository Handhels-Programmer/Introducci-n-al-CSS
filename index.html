Introduccion al CSS

Actividad 1

1. ¿Qué es CSS?

Es un lenguaje de hojas de estilo en cascada utilizado para controlar la presentación y el diseño de las páginas web escritas en HTML o XML.

2. ¿Cuál es la función principal de CSS?

Su función principal es separar el contenido de la estructura visual, permitiendo que los diseñadores controlen la apariencia de los elementos HTML, como colores, fuentes y disposición.

3. ¿Cómo se puede incluir CSS en un documento HTML?

Se puede incluir a través de estilos en línea dentro de las etiquetas HTML, estilos internos en la sección < head > con la etiqueta < style >, o estilos externos vinculando un archivo .css con la etiqueta < link >.

4. ¿Qué son los selectores en CSS?

Son patrones que permiten seleccionar elementos HTML específicos a los que se les aplicarán los estilos. Ejemplos incluyen selectores de tipo, de clase, de ID y selectores combinados.

5. ¿Qué son las propiedades y valores en CSS?

Las propiedades son características como color, font -size, margin, y los valores especifican cómo debe configurarse esa propiedad, como blue, 16px, 10px, respectivamente.

6. ¿Cómo se aplica el modelo de caja en CSS?

Define cómo se estructuran los elementos HTML como cajas rectangulares, que constan de contenido, relleno, borde y margen, afectando la disposición de los elementos en la página.

7. ¿Qué es el "cascading" en CSS?

Se refiere a cómo CSS aplica las reglas en función de su especificidad, importancia y el orden en el que están declaradas, permitiendo que las reglas más específicas o recientes sobrescriban a las anteriores.

8. ¿Qué son las media queries?

Se refiere a cómo CSS aplica las reglas en función de su especificidad, importancia y el orden en el que están declaradas, permitiendo que las reglas más específicas o recientes sobrescriban a las anteriores.

9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

Algunos estilos que se aplican frecuentemente son los colores (color, background -color), la tipografía (font-size, font-family), el espaciado (margin, padding), los bordes (border) y la alineación de texto y elementos (text-align, display).

10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

Se logra utilizando media queries, junto con diseños flexibles como Flexbox o CSS Grid, unidades relativas como porcentajes o vw/vh, y ajustando imágenes y textos para adaptarse automáticamente a distintos tamaños de pantalla.
